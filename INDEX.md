## üìå Most Used Commands

```bash
adb devices              # List devices
adb install app.apk     # Install APK
adb logcat              # View logs
adb push file /sdcard/  # Send file
adb pull /sdcard/file . # Get file
adb shell               # Shell access
adb shell screencap ... # Screenshot
```
